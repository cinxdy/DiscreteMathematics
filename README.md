# DiscreteMathematics Assignment 1  

## Problem Set  
p1 : Sudoku *  
p2 : Fill-a-Pix  
p3 : Numbrix  

## How to build?
<pre><code>
make p1  
make p2  
make p3  
</code></pre>
or
<pre><code>
gcc p1.c -o p1.out  
gcc p2.c -o p2.out  
gcc p3.c -o p3.out  
</code></pre>

## How to execute?
<pre><code>
./p1.out < input.txt  
./p2.out < input.txt  
./p3.out < input.txt  
</code></pre>
